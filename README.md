# Contractor Pro AI

All-in-one contractor app with real-time pricing + AI helper.

## Features

- **Price Comparison** - Compare prices from Lowe's, Home Depot, local stores
- **AI Bid Creator** - Tell AI what you need, get a professional bid in seconds
- **AI Pricing Advisor** - Ask anything about construction pricing
- **Store Locations** - Track local hardware stores with GPS
- **New Products Tracker** - Stay updated on new products
- **AI CEO** - Built-in AI that analyzes your business

## Tech Stack

- Python (Flask)
- AI (Groq/Qwen)
- Bootstrap 5
- Railway hosting
